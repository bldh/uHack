What it does


This is done through having two core parts to the project the IOS application and the website.

The IOS application provides the core function to our larges demographic market segments which is consists of tourists and hobbies/wildlife enthusiasts. This application is the way in which users collect, share and gain more information to education themselves and other about the wildlife that they are finding at there perticular locations. This is done by the users open the application and they are given a menu screen in which they have two options. The first is view collection, this view takes you to your collection or gallery of images. This has the ability to filter or share your selected images onto others. Selecting a single image itself from the view will take you to the photo screen that will show you a full screen image and more information about about the animal that has been found in the image. The second is Add a photo which will have a pop up window which will allow you to ether selected a pre existing image form the gallery or take a new photo.

The Website provides the core information for our second target market of researchers. Who may be interested in accessing the information that is collected and scored by the IOS application. Through a downloadable file to access the open sores database.
